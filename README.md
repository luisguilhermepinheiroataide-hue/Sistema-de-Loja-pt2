Sistema de Gerenciamento de Estoque

Desenvolvido inicialmente por Arthur Tabbi e Gabriela Oliveira

Este projeto tem como objetivo o desenvolvimento de um sistema de gerenciamento de estoque para uma loja, utilizando Java integrado a um banco de dados, com a finalidade de organizar, controlar e registrar as informações e movimentações de produtos. A proposta busca garantir maior precisão, eficiência e confiabilidade na gestão de estoque, permitindo um melhor acompanhamento das entradas e saídas de mercadorias.

O tema foi escolhido devido à sua relevância no contexto comercial, considerando que o controle de estoque é essencial para o bom funcionamento de qualquer empresa. Além disso, o projeto possibilita a aplicação prática dos conhecimentos adquiridos em banco de dados e programação orientada a objetos, sendo uma solução viável e adequada ao nível técnico atual dos desenvolvedores.

O sistema contará com funcionalidades como cadastro, edição e remoção de produtos, controle da quantidade em estoque, além do registro de movimentações de entrada e saída diretamente no banco de dados. Também será responsável por manipular dados como nome do produto, preço de venda, quantidade disponível e custo de aquisição (compra ou encomenda).

O projeto terá continuidade com os alunos Gabriela Oliveira e Luis Guilherme, que darão sequência ao desenvolvimento e aprimoramento das funcionalidades do sistema.
